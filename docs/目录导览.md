# 目录导览

```text
.
├── bin                 构建产物（CMake 输出可执行文件）
├── docs                项目文档
├── example
│   └── raftCoreExample 集群入口 raftCoreRun、示例客户端 callerMain
├── src
│   ├── common          通用工具、配置读取、日志和队列
│   ├── raftClerk       KV 客户端
│   ├── raftCore        Raft 与 KvServer 核心逻辑
│   ├── raftRpcPro      仅 .proto；生成代码在 cmake-build/.../generated
│   └── skipList        KV 状态机底层存储（头文件）
├── deploy              test.conf.fixed（固定端口，供脚本/CI）
├── scripts             cluster_up/down、smoke、bench_ci、smoke_chaos
└── test                kv_consistency / kv_bench
```

当前主链路已经统一为 gRPC；历史 mprpc 示例和自研 fiber 协程模块已从工程中移除。
