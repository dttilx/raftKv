# 目录导览

```text
.
├── bin                 构建产物和运行配置
├── docs                项目文档与图片
├── example
│   └── raftCoreExample Raft KV 集群启动和客户端示例
├── scripts             构建、启动、故障注入和压测脚本
├── src
│   ├── common          通用工具、配置读取、日志和队列
│   ├── raftClerk       KV 客户端
│   ├── raftCore        Raft 与 KVServer 核心逻辑
│   ├── raftRpcPro      gRPC / protobuf 接口定义
│   └── skipList        KV 状态机底层存储
└── test                一致性测试和压测入口
```

当前主链路已经统一为 gRPC；历史 mprpc 示例和自研 fiber 协程模块已从工程中移除。
