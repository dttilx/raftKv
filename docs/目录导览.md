```text
.
├── bin                 构建产物（CMake 输出可执行文件）
├── docs                项目文档
├── example
│   └── raftCoreExample 集群入口 raftCoreRun、示例客户端 callerMain
├── src

│   ├── raftRpcPro      仅 .proto；生成代码在 cmake-build/.../generated
│   └── skipList        KV 状态机底层存储（头文件）
├── deploy              test.conf.fixed（固定端口，供脚本/CI）
├── scripts             cluster_up/down、clean、smoke、bench_ci、bench_nightly、smoke_chaos

```

当前主链路已经统一为 gRPC；历史 mprpc 示例和自研 fiber 协程模块已从工程中移除。

## 核心分层

1. **Clerk**：客户端，向 KvServer 发起 Get / Put / Append。
2. **KvServer**：对外 gRPC KV 服务，写请求提交给 Raft。
3. **Raft**：选举、日志复制、提交、快照。
4. **raftRpcPro**：节点间与 KV 的 protobuf / gRPC 接口定义。
5. **skipList**：KV 状态机存储。
6. **common**：配置、工具、阻塞队列。

通信：客户端 ↔ KvServer（`kvServerRPC.proto`）；节点间（`raftRPC.proto`）；拓扑由 `NodeConfig` 读 `nodeXip` / `nodeXport`。
